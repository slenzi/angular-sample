Below are some direct links to our test RESTful service implemented using Jersey.

Data is pulled from CountryList.csv & Names.csv under src/main/resources/csv

People service. Data is pulled from Names.csv.

  • fetch person 1 (jersey/person/1)
  • fetch all people (jersey/person/all)

Sample service. Data is pulled from CountryList.csv.

  • fetch csv row count (jersey/sample/rowcount)
  • fetch data keyset as JSON (jersey/sample/keyset)
  • fetch data rows where first name starts with 'S' (jersey/sample/alpha/S)
  • fetch data rows where first name starts with 'S' and last name ends with 't' (jersey/sample/find?first=S&last=t)
